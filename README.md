# Planner App (Working Name: Chronicle)

A personal planning, habit tracking, and journaling application combining **calendars**, **events**, **structured logs**, and **templates**.  
The app helps users plan, track, and reflect on their activities with optional granular structure and automatically generated journals.

---

## Tech Stack

- **Backend:** Java Spring Boot  
- **Frontend:** React + Tailwind CSS  
- **Database:** SQLite3

---

## Features (V1)

- Multiple calendars per user  
- Hierarchical events and optional sub-tasks  
- Structured logs (text, number, duration, checkbox/list, table)  
- Notes tied to events and sub-tasks  
- Journal export with chronological, human-readable logs  
- Templates for reusable event structures
